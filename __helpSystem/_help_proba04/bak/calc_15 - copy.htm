π - число Пи = 3,14159265358979
d_ex - диаметр наружный, мм
d_in - диаметр внутренний, мм
b_ex - ширина наружная, мм
h_ex - высота наружная, мм
b_in - ширина внутренняя, мм
h_in - высота внутренняя, мм
S - площадь сечения, мм2
Wx - момент сопротивления осевой, мм3
Wp - момент сопротивления полярный, мм3
F - сила, N
L - длина, мм
M - момент силы, N*Metr
σ - напряжение нормальное, MPa
τ - напряжение касательное, MPa

5.1. Площадь сечения круга рассчитывается по формуле.

5.1.1. S = π * d_ex2 / 4

5.2. Площадь сечения круглой трубы.

5.2.1. S = ( π * d_ex2 / 4) - ( π * d_in2 / 4)

5.3. Площадь сечения прямоугольника.

5.3.1. S = b_ex * h_ex

5.4. Площадь сечения прямоугольной трубы.

5.4.1. S = (b_ex * h_ex) - (b_in * h_in)

5.5. Момент сопротивления круга осевой.

5.5.1. Wx = π * d_ex3 / 32

5.6. Момент сопротивления круга полярный.

5.6.1. Wp = π * d_ex3 / 16

5.7. Момент сопротивления круглого кольца осевой.

5.7.1. Wx = ( π * d_ex3 / 32) * (1.0 - (d_in / d_ex)4)

5.8. Момент сопротивления круга полярный.

5.8.1. Wp = ( π * d_ex3 / 16) * (1.0 - (d_in / d_ex)4)

5.9. Момент сопротивления прямоугольника осевой.

5.9.1. Wx = (b_ex * h_ex2) / 6.0

5.10. Момент сопротивления прямоугольника полярный.

5.10.1. Wp = alfa(h_ex/b_ex) * h_ex * b_ex2

5.11. Момент сопротивления прямоугольной трубы осевой.

5.11.1. Wx = ( (b_ex * h_ex3) - (b_in * h_in3)) / (6.0 * h_ex)

5.12. Момент сопротивления прямоугольной трубы полярный.

5.12.1. Wp = Фиг посчитаешь

5.13. Момент силы. (изгиба)

5.13.1. M = F * L

5.14. Напряжение в детали

5.14.1. При растяжении... σ = F / S
5.14.2. При изгибе............ σ = M / Wx
5.14.3. При кручении....... τ = M / Wp
5.14.4. При срезе............... σ = F / S
